FoodPlanetVR
============
>Oculus Quest2 환경의 VR 요리 시뮬레이션 게임   
>>*'궁핍한 지구생활을 정리하고 우주로 떠나    
>>외계 손님들을 상대로 식당을 열어보자!'*


## 게임 개요

### 테스트 환경
- Device - Facebook 사 Oculus Quest 1/2   
- Minimum SDK - Android 4.4 KitKat(API Level 19)
